# homebrew-markview

Homebrew tap for [markview](https://github.com/ericbrophy/markview), a cross-platform desktop markdown viewer/editor.

## Install

```bash
brew install --cask ericbrophy/markview/markview
```

The `<owner>/<repo>/<cask>` triple form auto-taps and installs in one step. No prior `brew tap` needed.

## Update

```bash
brew upgrade --cask markview
```

## Uninstall

```bash
brew uninstall --cask markview
```

This removes the app. To also remove user data and settings:

```bash
brew uninstall --zap --cask markview
```

## Requirements

- macOS 11 (Big Sur) or later
- Apple Silicon or Intel — the cask installs a universal binary
